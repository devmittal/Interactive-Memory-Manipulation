09/25: This program, on execution, first displays a greeting with a range of options from which the user can select. These options are separate functions in different C source files. Currently these functions do not have any functionality and are just dummy files. They have been compiled and linked together using the makefile functionality.

10/03: All the functionalities have been added to each function. This is an interactive memory manipulation command line utility. The user can allocate memory, write data to the allocated memory, display the data in the allocated memory, invert data bits in specified memory locations, write a pseudo random pattern to memory locations and verify this pseudo random pattern. All the source files and header files have been compiled used make feature. It also contains an automated testing feature which tests every function automatically.
